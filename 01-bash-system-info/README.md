# EC2 System Health Monitoring Script

## Project Overview
A Bash script for automated system health monitoring on AWS EC2 instances. This tool generates detailed reports and maintains a concise log for trend analysis—essential skills for a Linux Support Engineer.

## Objectives
- Automate routine system checks
- Demonstrate proficiency in Bash scripting and cron scheduling
- Implement logging best practices
- Set up threshold-based monitoring
- Create production-ready documentation

## Skills Demonstrated
- **Bash Scripting**: Variables, conditionals, command substitution
- **System Monitoring**: Disk, memory, process, and service checks
- **Automation**: Cron job scheduling
- **Logging**: Dual-file strategy (detailed reports + concise logs)
- **AWS EC2**: Cloud-specific environment management
